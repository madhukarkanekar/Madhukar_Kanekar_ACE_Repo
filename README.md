# Oracle Fusion HCM REST API Integration using PL/SQL

This repository contains a sample PL/SQL package that demonstrates how to call Oracle Fusion HCM REST APIs using UTL_HTTP and process JSON responses inside the Oracle database.

The package simulates a real-time integration scenario where worker and manager data is retrieved from Oracle Fusion HCM and processed for validation and staging table updates.

---

## Product Used

- Oracle Fusion Cloud HCM
- HCM REST API
- PL/SQL
- UTL_HTTP
- JSON_OBJECT_T / JSON_ARRAY_T
- SQL
